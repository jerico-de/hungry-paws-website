# Pet Care Management System

A responsive multi-page website for a pet care business, featuring grooming and hotel services, customer testimonials, and an interactive contact form.  
Built using HTML, CSS, JavaScript, and Bootstrap for a clean, modern, and mobile-friendly design.

## Features

- Pet Grooming & Hotel Services with responsive grid layout.  
- Image Carousel for promotions, shop items, and testimonials.  
- Contact Form with working feedback modal.  
- Thank You Modal with functional close button.  
- Fully responsive layout with consistent design across all pages.

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS + Bootstrap 5 components)  
- Bootstrap 5 (for responsive grid and modals)

## Pages

- `index.html` – Homepage with carousel and featured sections.  
- `services.html` – Pet grooming and hotel services with grid layout.  
- `contact.html` – Contact form with thank-you modal.  
- `about.html` – Company information and mission.
